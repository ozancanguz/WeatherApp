# WeatherApp

- I have developed a weather mobile app. We can search 
city on Turkey and app displays weather for selected 
city.


**Tools that I used** :

- Navigation Component
- Hilt for dependency injection
- Retrofit 
- Postman for Api testing.
- Livedata.
- View Binding
- Glide for image loading
- RESTApi
- LottieAnimationView

**Arhictecture:**

- MVVM
